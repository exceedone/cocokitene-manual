## Exment とは

Cocokitene は、ブロックチェーン技術を利用し、総会情報を効率的に保存する Web です
<a href="" target="_blank">Cocokitene 公式サイト</a>

## 本マニュアルについて

本マニュアルでは、常に最新版を想定したマニュアルとなっております。  
マニュアルに記載の機能やコマンドなどは、バージョンアップにより追加されたものでも、そのバージョンの記載が無い場合があります。あらかじめご了承ください。
